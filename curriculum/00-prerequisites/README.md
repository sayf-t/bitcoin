# Module 00: Prerequisites and Fundamentals

## Overview
This module covers the essential knowledge and skills required before diving into Bitcoin Core development. It ensures you have the necessary foundation in programming languages, design patterns, tools, and core concepts.

## Learning Objectives
After completing this module, you will:
- Understand modern C++ programming concepts essential for Bitcoin Core
- Be proficient in Python for testing and tooling
- Know key design patterns used throughout the codebase
- Be comfortable with development tools and workflows
- Grasp fundamental cryptographic concepts
- Understand networking principles crucial for P2P systems

## Module Structure

- STL containers and algorithms

### [2. Python Fundamentals](./02-python-fundamentals/README.md)

### [3. Design Patterns](./03-design-patterns/README.md)

### [6. Networking Fundamentals](./06-networking-fundamentals/README.md)
