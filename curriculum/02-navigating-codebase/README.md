# Module 2: Navigating the Bitcoin Core Codebase

## Overview
This module teaches effective navigation and understanding of the Bitcoin Core codebase structure.

## Topics Covered
1. Source code organization
2. Code navigation techniques
3. Feature location strategies
4. Documentation structure
5. Development tools and utilities

## Learning Objectives
- Understand Bitcoin Core's codebase organization
- Master code navigation tools and techniques
- Learn effective feature location strategies
- Navigate and utilize project documentation
- Use development tools efficiently

## Projects and Exercises
- Code Explorer Project
- Feature Location Exercise
- Documentation Navigation
- Tool Usage Workshop
