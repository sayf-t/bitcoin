# Contribution Workflow Exercises

## Exercise 1: Repository Setup
1. Fork the Bitcoin Core repository
2. Clone your fork locally
3. Set up upstream remote
4. Create a feature branch
5. Practice keeping your branch updated

## Exercise 2: Code Style and Standards
1. Find and read the Bitcoin Core style guide
2. Set up code formatting tools
3. Format a file according to guidelines
4. Run linting checks
5. Fix any style violations

## Exercise 3: Making Changes
1. Find a "good first issue"
2. Create a feature branch
3. Make the necessary changes
4. Ensure code style compliance
5. Add appropriate documentation

## Exercise 4: Testing
1. Run the full test suite
2. Create new test cases
3. Run specific test categories
4. Debug test failures
5. Measure test coverage

## Exercise 5: Commit Management
1. Create well-formatted commits
2. Practice interactive rebasing
3. Amend commits
4. Squash related commits
5. Write good commit messages

## Exercise 6: Pull Request Creation
1. Prepare your branch for PR
2. Create a PR with proper description
3. Fill out the PR template
4. Add appropriate labels
5. Link related issues

## Exercise 7: Code Review Process
1. Review an existing PR
2. Provide constructive feedback
3. Respond to review comments
4. Update PR based on feedback
5. Handle CI failures

## Exercise 8: Community Interaction
1. Join the developer IRC channel
2. Participate in a PR review club
3. Subscribe to the mailing list
4. Engage in technical discussions
5. Help other contributors

## Exercise 9: Documentation
1. Improve existing documentation
2. Add inline code comments
3. Update API documentation
4. Write clear commit messages
5. Create user guides

## Exercise 10: Advanced Workflows
1. Backport a commit
2. Cherry-pick changes
3. Resolve merge conflicts
4. Handle complex rebases
5. Manage multiple PRs

## Submission Guidelines
- Document your workflow
- Include command history
- Explain your decisions
- Show before/after states
- Provide feedback on the process

## Advanced Challenges
1. Create a contribution guide
2. Develop workflow scripts
3. Improve CI processes
4. Create PR templates
5. Build review checklists
